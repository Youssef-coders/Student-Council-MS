What We Do

The Programming Club is designed for students passionate about coding and technology. Learn to build apps, websites, games, and more while developing problem-solving skills.

Activities Include:

  • Coding workshops and tutorials
  • Group project development
  • Programming language courses (Python, JavaScript, etc.)
  • Hackathons and coding competitions
  • Guest speakers from tech industry
  • Web development and app creation

Who Can Join?

Everyone interested in programming! No prior experience needed. We'll start from the basics and advance together.
